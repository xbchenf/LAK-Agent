办公时间:  工作日 9:00-12:00, 14:00-17:00

一、办理条件

公民、法人或者其他组织认为行政机关的具体行政行为侵犯其合法权益的，均可以申请行政复议。申请应当符合以下条件：
1. 有明确的申请人和被申请人；
2. 有具体的行政复议请求和事实依据；
3. 属于行政复议范围；
4. 在法定申请期限内（知道具体行政行为之日起60日内）。

二、所需材料

（一）申请人需提供以下材料：
1. 行政复议申请书一式两份（应载明申请人的基本情况、被申请人名称、行政复议请求、事实与理由、申请日期）；
2. 申请人身份证明（自然人提供身份证复印件，法人提供营业执照复印件并加盖公章）；
3. 被复议的具体行政行为的法律文书复印件（如行政处罚决定书、行政许可决定书等）；
4. 委托他人代理的，应当提交授权委托书及代理人身份证明；
5. 支持行政复议请求的证据材料（如有）。

（二）行政复议申请书写要点：
1. "申请人"部分——写明姓名、性别、身份证号、地址、联系电话；
2. "被申请人"部分——写明行政机关全称、法定代表人、地址；
3. "请求"部分——写明要求撤销、变更或确认违法的具体行政行为；
4. "事实与理由"部分——按时间顺序陈述事实，说明法律依据。

三、办理流程

（一）提交申请：
1. 申请人可以选择以下方式提交申请：
   - 现场提交：携带材料到省政务服务中心三楼行政复议窗口；
   - 邮寄提交：将材料邮寄至省司法厅行政复议处；
   - 网上提交：登录"XX省行政复议在线服务平台"上传材料；
2. 窗口工作人员进行形式审查，材料齐全的当场受理并出具受理通知书；
3. 材料不齐全的，当场或5日内一次性告知需要补正的内容。

（二）审理：
1. 行政复议机关受理后，7日内将申请书副本发送被申请人；
2. 被申请人应当自收到之日起10日内提出书面答复并提交证据材料；
3. 申请人可以到复议机关查阅被申请人提交的材料（涉及国家秘密、商业秘密、个人隐私的除外）；
4. 复议机关依法进行书面审查，必要时调查取证。

（三）作出决定：
1. 行政复议机关自受理之日起60日内作出行政复议决定；
2. 情况复杂的可延长30日，并通知申请人；
3. 申请人对复议决定不服的，可以在收到决定书之日起15日内向人民法院提起行政诉讼。

四、收费标准

行政复议申请不收取任何费用。

五、注意事项

1. 行政复议期间具体行政行为不停止执行，但被申请人或复议机关认为需要停止执行的除外；
2. 申请人撤回行政复议申请的，不得再以同一事实和理由申请复议；
3. 申请人对行政复议决定不服的，应在收到决定书之日起15日内向人民法院起诉；
4. 如对行政复议有疑问，可拨打咨询电话029-87654322或到窗口现场咨询。
